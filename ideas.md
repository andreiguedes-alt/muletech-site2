## Ideia 1 – "Circuito Vital" (Biomecânica Digital)

**Design Movement:** Brutalist Tech Noir com influências de UI futurista biomédica

**Core Principles:**
1. Contraste extremo entre escuridão e luz — o dispositivo é uma faísca de vida no escuro
2. Tipografia monumental com tracking expandido, evocando precisão clínica
3. Linhas de circuito como linguagem visual — conectividade e fluxo de dados como metáfora
4. Revelação progressiva — cada scroll desvela uma camada do sistema

**Color Philosophy:**
- Fundo: #050A14 (quase preto com toque azul)
- Primário: #0A1628 (azul marinho profundo)
- Secundário: #1E3A5F (azul médio)
- Acento: #FF6B2B (laranja vivo para alertas)
- Highlight: #00D4FF (ciano elétrico para dados e conexões)
- Texto: #E8F4FD (branco azulado)

**Layout Paradigm:**
- Scroll horizontal em seções específicas (como Tesla)
- Seções em tela cheia com snap-scrolling
- Elementos que "emergem" do fundo escuro ao scroll
- Linhas de circuito animadas conectando seções

**Signature Elements:**
1. Linhas de circuito PCB animadas como separadores e backgrounds
2. Números de dados em tempo real (simulados) mostrando métricas do sistema
3. Partículas de dados fluindo entre seções

**Interaction Philosophy:**
- Hover revela informações técnicas ocultas
- Cursor customizado com efeito de "scan"
- Botões com efeito de "power on" ao clicar

**Animation:**
- Entrada de texto com efeito de digitação (typewriter)
- Elementos surgem com fade + slide de baixo para cima
- Linhas de circuito se "desenham" ao scroll
- Parallax profundo em camadas múltiplas

**Typography System:**
- Display: Space Grotesk Bold (700) — industrial, técnico
- Body: DM Sans Regular (400) — legível, moderno
- Mono: JetBrains Mono — dados técnicos e métricas
- Hierarquia: 96px / 64px / 48px / 32px / 18px / 14px

## Ideia 2 – "Movimento Seguro" (Editorial Cinético)

**Design Movement:** Swiss International Style com motion design contemporâneo

**Core Principles:**
1. Grid assimétrico quebrando a monotonia — tensão visual controlada
2. Tipografia como elemento visual principal, não apenas conteúdo
3. Fotografia editorial de alto contraste integrada ao layout
4. Movimento como narrativa — o scroll conta a história do produto

**Color Philosophy:**
- Fundo: #FFFFFF com seções em #F0F4F8
- Primário: #0B1F3A (azul marinho)
- Secundário: #1A4B8C (azul médio vibrante)
- Acento: #E85D04 (laranja queimado — urgência, alerta)
- Neutro: #6B7280 (cinza para texto secundário)

**Layout Paradigm:**
- Colunas assimétricas (7/5, 3/9) quebrando o grid regular
- Texto grande invadindo imagens
- Seções alternando entre fundo branco e azul marinho
- Timeline horizontal para "Como Funciona"

**Signature Elements:**
1. Números grandes (01, 02, 03) como elementos decorativos de seção
2. Linhas diagonais e cortes angulares entre seções
3. Badges e tags técnicas flutuantes sobre imagens

**Interaction Philosophy:**
- Scroll suave com momentum
- Cards com elevação ao hover
- Botões com preenchimento de cor animado

**Animation:**
- Texto dividido por palavras com stagger animation
- Imagens com clip-path reveal ao scroll
- Contadores numéricos animados nas métricas

**Typography System:**
- Display: Syne ExtraBold (800) — impacto editorial
- Body: Inter Regular (400) — clareza máxima
- Accent: Syne Mono — detalhes técnicos
- Hierarquia: 80px / 56px / 40px / 28px / 16px / 13px

## Ideia 3 – "Proteção Luminosa" (Cinematic Dark Tech) ← ESCOLHIDA

**Design Movement:** Cinematic Dark Tech — fusão de premium consumer tech (Apple) com urgência visual (Tesla/Rockstar)

**Core Principles:**
1. Escuridão como palco — elementos luminosos emergem do fundo escuro com impacto máximo
2. Tipografia monumental e espaçada — cada palavra tem peso e intenção
3. Narrativa visual progressiva — o scroll é uma jornada cinematográfica
4. Laranja como sinal de vida — usado cirurgicamente para alertas e CTAs críticos

**Color Philosophy:**
- Fundo: #070D1A (azul-preto profundo, evoca noite e segurança)
- Primário: #0D1F3C (azul marinho rico)
- Secundário: #1B3A6B (azul médio para cards e seções)
- Acento: #FF6600 (laranja puro — alerta, urgência, vida)
- Glow: #4A9EFF (azul elétrico para efeitos de luz e destaques)
- Texto: #F0F8FF (branco azulado suave)
- Texto secundário: #8BA8C8 (azul acinzentado)

**Layout Paradigm:**
- Hero em tela cheia com parallax profundo
- Seções de largura total com conteúdo centralizado mas assimétrico internamente
- "Floating cards" com glassmorphism para features técnicas
- Linha do tempo vertical animada para "Como Funciona"

**Signature Elements:**
1. Efeito de "glow" azul/laranja em elementos chave (botões, ícones, bordas)
2. Partículas flutuantes sutis no background do hero
3. Cards com borda luminosa e backdrop-blur (glassmorphism)

**Interaction Philosophy:**
- Scroll revela elementos com animações suaves e impactantes
- Hover em cards acende o glow
- Botão de simulação de queda com animação dramática

**Animation:**
- Hero: texto entra com split animation por letras
- Seções: fade-in + translate-y ao entrar no viewport
- Cards: scale + glow ao hover
- Simulação de queda: sequência animada com estados visuais

**Typography System:**
- Display: Barlow Condensed ExtraBold (800) — impacto industrial, cinematic
- Subheading: Barlow SemiBold (600) — hierarquia clara
- Body: Barlow Regular (400) — leitura confortável
- Mono: Space Mono — dados técnicos
- Hierarquia: 96px / 72px / 48px / 32px / 20px / 16px
</text>
